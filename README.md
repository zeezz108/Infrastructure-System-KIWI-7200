# Infrastructure Diagnostic System

Централизованная система для диагностики инфраструктуры. Агенты получают задачи от сервера, выполняют проверки и возвращают результаты.

## Быстрый старт

### 1. Запуск сервера
```bash
python backend/main_simple.py
```

Сервер запустится на http://localhost:8000

Логин: admin

Пароль: admin123

### 2. Добавление агента через веб
Открой http://localhost:8000/agents.
Нажми "+ Добавить агента".
Введи имя.
Нажми "Зарегистрировать".

### 3. Запуск агента
Открой http://localhost:8000/agents.

1 вариант (через Web):
Нажми "Запустить".

2 вариант (через Terminal): Нажми "Команда" напротив нужного агента, затем "Копировать". 
Вставь команду в Terminal и запусти.

### 4. Создание задачи
На странице агентов выбери агента.
Выбери тип проверки.
Нажми "Запустить".

### 5. Функционал
1) Регистрация/Запуск/Остановка/Удаление агентов через веб
2) Рейтинг агентов
3) Очередь задач
4) Шаблоны проверок 
5) Экспорт задач в JSON/CSV 
6) Метрики и графики 
7) Хранение истории в SQLite 
8) Real-time обновление данных
9) Сравнение результатов задач 
10) Ограничение количества одновременных задач на агенте (2 задачи)
11) Визуализация связей
